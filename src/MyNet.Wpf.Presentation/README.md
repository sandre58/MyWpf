## Installation

Install via NuGet:

```bash
dotnet add package MyNet.Wpf.Presentation
```

## Features

- Controls and helpers for MVVM and presentation logic
- Selectors, converters, and parameters
- Custom views and styles

## Example Usage

```csharp
// Example: Use a selector
```

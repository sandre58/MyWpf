## Installation

Install via NuGet:

```bash
dotnet add package MyNet.Wpf.Web
```

## Features

- Controls and helpers for web content in WPF
- Custom views and resource integration

## Example Usage

```csharp
// Example: Show a web view
var webView = new MyWebView();
```
